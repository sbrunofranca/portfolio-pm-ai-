# 📊 AI Analytics Copilot (NL → SQL)

---

## 🚨 Problema

Usuários de negócio não conseguem acessar dados sem depender de:

- SQL
- BI dashboards
- Analistas

---

## 🎯 Solução

Sistema que converte perguntas em linguagem natural em consultas SQL automaticamente.

---

## 🧠 Funcionalidade

- Pergunta em linguagem natural
- Geração automática de SQL via LLM
- Execução em banco de dados
- Retorno estruturado

---

## ⚙️ Stack

- Python
- Streamlit
- SQLAlchemy
- Groq LLM
- SQLite

---

```bash
pip install -r requirements.txt
