# Política de Entregas

## Objetivo
Definir padrões de entrega para operações de e-commerce.

## Métodos de Entrega
- Entrega padrão.
- Entrega expressa.
- Retirada em loja.
- Entrega agendada.

## Prazo de Entrega
Os prazos variam conforme:
- Região de entrega.
- Tipo de frete escolhido.
- Disponibilidade de estoque.

## Rastreamento
- Todos os pedidos devem possuir código de rastreio.
- Atualizações automáticas devem ser enviadas ao cliente.

## Problemas Comuns
### Pedido Atrasado
- Verificar status com a transportadora.
- Atualizar cliente com previsão revisada.

### Pedido Extraviado
- Abrir chamado com transportadora.
- Oferecer reenvio ou reembolso.

### Pedido Entregue Incorretamente
- Validar comprovante de entrega.
- Iniciar processo de investigação.

## Tentativas de Entrega
- Até 3 tentativas de entrega.
- Após falha, pedido retorna ao centro logístico.

## Responsabilidades
### Loja
- Embalar corretamente os produtos.
- Informar prazos reais de entrega.

### Transportadora
- Garantir transporte seguro.
- Atualizar eventos logísticos.

### Cliente
- Informar endereço correto.
- Garantir disponibilidade para recebimento.

## Indicadores Operacionais
- SLA de entrega.
- Taxa de atraso.
- Taxa de extravio.
- Tempo médio de entrega.
